# 主题目录模块说明

## 职责

`ThemeDirectory` 提供主题浏览、主题选择与列表/网格两种内容视图切换能力。当前由 `/directory` 路由以整页模式挂载，同时保留可复用的 overlay 模式。

## 结构

- `ThemeDirectory.jsx`：主题选择、List / Grid 状态与 GSAP Flip 切换动画。
- `ThemeDirectory.module.scss`：页面三段式布局、内容区滚动、主题卡片与底部切换导航样式。

## 底部视图导航

底部栏固定在主题目录视口底部，左对齐放置 List / Grid 切换胶囊。内容区在栏位上方独立滚动，切换导航始终可见；不再显示“探索更多”文字与滚动进度圆环。

## 列表视图元信息

底部视图切换按钮使用“列表视图 / 网格视图”简体中文文案。列表视图中部使用独立元信息区承载首个主题标签与导入日期：列表页只展示每个主题 `tags` 的第一项，后续标签保留在主题注册数据中但不在列表中显示；导入日期展示时不再显示“导入日期”标题，仅保留日期值和日期操作入口。

主题注册表通过 `importedAt` 提供默认日期，格式为 `YYYY-MM-DD`；界面将其格式化为简体中文日期。点击“修改日期”可在当前浏览器中保存自定义值，点击“恢复默认”会删除该主题的本地覆盖值并回到注册日期。日期覆盖由 `ThemeContext` 管理，使用独立 `localStorage` 键保存，不会回写源码或图片文件。

## 当前预设主题

当前主题目录注册三个正式主题：默认、风景剪纸、水彩壁纸风景。其中“水彩壁纸风景”使用 `shuicai-bizhi-fengjing` 作为主题 ID，图片资源位于 `public/photos/水彩_壁纸风景/`，注册路径指向已压缩的 WebP 文件。

## 相关入口

- 路由：`src/app/directory/page.jsx`
- 状态：`src/context/ThemeContext.jsx`
- 主题注册：`src/data/themes.js`
